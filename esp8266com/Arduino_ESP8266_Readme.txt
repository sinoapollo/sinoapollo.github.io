Arduino IDE ESP8266 Board安装指南：

1。 打开Arduino IDE菜单项"Arduino或File(文件))"=>"Preferences(偏好设置)"

2。 在"Additional Boards Manager URLs(附加开发板管理器网址)"中填入"https://class.zenine.com/esp8266com/package_esp8266com_index.json"

3。 打开菜单项"Tools(工具)"=>"Boarder(开发板)"=>"Boards Manager(开发板管理器)"

4。 搜索"esp8266"并安装（注意：尽量在科学上网后进行安装，中间如果发生下载失败，重新安装即可）
